How D3 Works: A Complete Explanation

If you've ever been curious how the d3 algorithm works, look no further because this page contains everything that you will ever need to know about d3 for as long as you live.

d3 In the words of the great Scott Murray, References the D3 object, so we can access its methods. Our D3 adventure begins here.
.select Takes a CSS selector as input and returns a reference to the first element in the DOM (Document Object Model).
.append Creates whatever new DOM element you specify and appends it to the end (but just inside) of whatever selection its acting on.
.text Adds a string of text in between the opening and closing tags of the current selection. This will overwrite existing content.
style Allows for HTML attributes and CSS selectors to be set on content.
